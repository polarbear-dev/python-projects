# Purchase data

The data is taken from here: https://www.kaggle.com/datasets/ulrikthygepedersen/online-retail-dataset

The table includes the following fields:

- **InvoiceNo** — Purchase identifier
- **StockCode** — Stock item identifier
- **Description** — Textual description of the item
- **Quantity** — Quantity of the item
- **InvoiceDate** — Purchase date
- **UnitPrice** — Price of the item
- **CustomerID** — Customer identifier
- **Country** — Country
